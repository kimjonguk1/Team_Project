# 프로젝트 주의 사항
- 프로젝트 주제 선정(완료, 영화)
- 구현 기능 정리(가능하면 /대상/동작, 불가피한 대상은 예외).
- 역할 분담.(완료)
- ER 구조 정리(구현 기능에 따라 스키마와 테이블, 제약 조건 구조 정리) 및 데이터베이스 구축, 엔티티 클래스 구현
- 디자인 패턴 확립 < 조장은 조원이 디자인 패턴을 벗어나지 않는지 확인할 것.
    - 반응형 구현 여부 정하기
    - 모바일 페이지 구현 여부 정하기
    - 대표 색상 1가지 ~ 5가지 정하기
    - 대표 폰트 정하기(완료, 프리텐다드)
    - 로고 정하기(구글에 Free Logo Maker 검색하면 많이 나옴).(완료, JGV)
